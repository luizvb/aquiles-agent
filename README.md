# Aquiles Agent

The sales site for [Aquiles](https://agent.netolabs.dev), NetoLabs' end-to-end AI development agent for Codex.

Aquiles covers the full product lifecycle: discovery, architecture, implementation, testing, security guardrails, LGPD-aware workflows, PII redaction, deployment, observability, and continuous improvement.

## Stack

- Next.js 16 with App Router
- React 19
- Tailwind CSS 4
- Motion
- Phosphor Icons

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run build
```

## Product

Aquiles is delivered as a plug-and-play Codex plugin. The landing page source is open under MIT; the commercial Aquiles agent is sold separately for USD 1,999.
